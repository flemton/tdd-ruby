# ğŸ“– The Solver App <a name="about-project"></a>

**The Solver App** is a Ruby application that provides a Solver class with three methods: factorial, reverse, and fizzbuzz. This project follows the principles of Test-Driven Development (TDD) using RSpec for testing.
